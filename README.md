# IoT BLE Flutter App

Application mobile Flutter pour communiquer en BLE avec un STM32 NUCLEO-WB55 via le protocole Blue-ST.

## Fonctionnalités

- Scan des appareils BLE à proximité
- Connexion au service Blue-ST
- Affichage température en temps réel (notifications)
- Contrôle LED à distance

## Prérequis

- Flutter 3.38+
- Android SDK
- Téléphone Android avec BLE

## Installation
```bash
flutter pub get
flutter run
```

## Permissions Android

L'application nécessite les permissions Bluetooth et Localisation.
